# 项目总览

本文件说明：从开题报告抽取项目定位、核心模块和落地顺序，帮助团队在 VSCode 中统一认知。

## 对应开题报告的核心模块
1. 项目与插件管理
2. OpenClaw Skill 专项审计
3. 通用依赖解析与 SBOM 生成
4. 多源漏洞库匹配与扫描
5. 风险可视化分析
6. 智能修复指引
7. 审计报告生成与导出

## 分层设计
- 前端交互层：`frontend/`
- 后端服务层：`backend/app/api` `backend/app/services`
- 核心引擎层：`backend/app/engine`
- 数据存储层：`backend/app/db` `backend/app/models` `data/`

## 迭代建议
- 先做“上传 -> 扫描 -> 查看结果”的闭环。
- 再做可视化大屏与报告导出。
- 最后接入 CNVD/CNNVD/NVD/OSV 聚合与更强规则系统。
