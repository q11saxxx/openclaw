# 数据库草图

本文件说明：给出开发阶段 SQLite 与部署阶段 MySQL 都能复用的表结构草图。

## 核心表
- `projects`：项目信息
- `artifacts`：上传的插件包或依赖清单
- `scan_tasks`：扫描任务
- `scan_findings`：自研规则命中项
- `sbom_documents`：SBOM 原始结果
- `vulnerability_items`：漏洞匹配结果
- `report_exports`：导出报告记录

## 扩展点
后续可新增：`users`、`audit_logs`、`rule_versions`、`fix_suggestions`。
