## 🚀 快速开始

### 方式一：使用启动脚本（推荐）

**Windows:**
```bash
双击 start-frontend.bat
```

**Linux/Mac:**
```bash
chmod +x start-frontend.sh
./start-frontend.sh
```

### 方式二：手动启动

```bash
cd frontend
npm install
npm run dev
```

访问: http://localhost:3000

## 🎯 功能演示路径

### 完整审计流程
1. **上传Skill**: Dashboard → 点击"上传Skill" → 选择文件 → 上传
2. **查看Skill**: Skills页面 → 点击"查看详情"
3. **发起审计**: Skill详情 → 点击"开始审计"
4. **查看进度**: 自动跳转到进度页面
5. **查看报告**: 审计完成 → 点击"查看报告"
6. **下载报告**: 报告详情 → 点击"导出MD/JSON"

### 规则管理流程
1. **查看规则**: Rules页面 → 查看内置规则
2. **创建规则**: 点击"创建规则" → 填写表单 → 保存
3. **编辑规则**: 点击"编辑" → 修改内容 → 更新
4. **删除规则**: 点击"删除" → 确认 → 删除成功
